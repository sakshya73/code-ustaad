# Privacy Policy

## Code Ustaad Extension

**Last Updated:** January 2025

### What Data We Collect

Code Ustaad collects and processes the following data:

1. **Selected Code Snippets** - The code you highlight and ask Ustaad to explain
2. **API Keys** - Your OpenAI or Gemini API keys (stored locally and encrypted)

### How We Use Your Data

- **Code Snippets**: Sent directly to your chosen AI provider (OpenAI or Google Gemini) solely for generating explanations. We do not store, log, or transmit your code to any other server.

- **API Keys**: Stored securely on your local machine using VS Code's encrypted SecretStorage. Keys are never transmitted to our servers.

### Third-Party Services

This extension uses the following third-party APIs based on your configuration:

| Provider | Privacy Policy |
|----------|---------------|
| OpenAI | [https://openai.com/privacy](https://openai.com/privacy) |
| Google Gemini | [https://ai.google.dev/terms](https://ai.google.dev/terms) |

Your code is sent directly to these providers using **your own API key**. We act only as a client interface.

### Data Storage

- **Local Only**: All history and settings are stored locally on your machine
- **No Server**: Code Ustaad does not have any backend server
- **No Telemetry**: We do not collect analytics, telemetry, or usage data

### Your Rights

- You can clear your API keys anytime using `Code Ustaad: Clear API Key`
- You can clear your history using the "Clear" button in the History panel
- Uninstalling the extension removes all locally stored data

### Contact

For privacy concerns, please open an issue at:
[https://github.com/sakshya73/code-ustaad/issues](https://github.com/sakshya73/code-ustaad/issues)
